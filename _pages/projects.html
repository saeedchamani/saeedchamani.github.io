--- layout: archive title: "Related Course Projects" permalink: /projects/ author_profile: true --- ## Image and Video Processing: - Developed a [**neural image colorization**](https://github.com/nikaemami/Neural-Image-Colorization-using-ResNet) model using ResNet-18 with perceptual and pixel-wise loss for grayscale-to-RGB translation. - Implemented [**adaptive image segmentation**](https://github.com/nikaemami/Adaptive-Image-Segmentation) using histogram clustering and region growing. - Compared [**Mean Shift**](https://github.com/nikaemami/Mean-Shift-vs-Graph-Cut-Image-Segmentation) and [**Graph Cut**](https://github.com/nikaemami/Mean-Shift-vs-Graph-Cut-Image-Segmentation) segmentation methods across diverse image types. - Applied classical CV techniques: [**template matching**](https://github.com/nikaemami/Template-Matching), [**thresholding** (Otsu, Niblack)](https://github.com/nikaemami/Image-Thresholding), and [**histogram equalization**](https://github.com/nikaemami/Enhancing-Image-Contrast-Using-Histogram-Equalization-Techniques). - Built feature alignment pipelines with [**Harris corner detection**](https://github.com/nikaemami/Harris-Detector-and-SIFT-Descriptor), [**SIFT descriptors**](https://github.com/nikaemami/Harris-Detector-and-SIFT-Descriptor), and [**panorama stitching**](https://github.com/nikaemami/Panorama-Stitching) using RANSAC. - Evaluated [**image denoising**](https://github.com/nikaemami/Image-Denoising-with-Average-and-Gaussian-Filters) methods (Gaussian and average filters) using PSNR under varied noise levels. - Constructed [**Gaussian and Laplacian pyramids**](https://github.com/nikaemami/Gaussian-and-Laplacian-Pyramids) for multi-scale image decomposition and reconstruction. - Explored color and spatial processing by [**analyzing RGB/HSV color channels**](https://github.com/nikaemami/Analyzing-Color-Channels-for-RGB-and-HSV-Images) and [**applying 2D convolutions**](https://github.com/nikaemami/Analyzing-Filter-Effects-in-Spatial-and-Frequency-Domains) with various filters. - Developed a [**block-based hybrid video coder**](https://github.com/nikaemami/Block-Based-Hybrid-Video-Coding) using integer-accuracy EBMA for P-frame compression with intra/inter prediction. ## Audio & Speech Processing: - Developed a [**sound event classification**](https://github.com/nikaemami/Sound-Event-Classification) pipeline using log-Mel spectrograms from the ESC-50 dataset. Compared traditional models (SVM, Random Forest) and neural networks (MLP, Conv1D), and evaluated performance using temporal pooling strategies. - Developed a [**voice activity detection**](https://github.com/nikaemami/Voice-Detection) system using neural networks and log-Mel spectrogram features to identify speech segments in noisy audio. - Built a multimodal [**audio captioning**](https://github.com/nikaemami/LLM-Automated-Audio-Captioning) pipeline by extracting mel-spectrogram and wavelet-based features, and generating descriptive captions using a pre-trained [**Vicuna LLM**](https://github.com/nikaemami/LLM-Automated-Audio-Captioning). - Explored [**fundamental audio features**](https://github.com/nikaemami/Sound-Processing-Fundamentals) such as envelope, energy, spectral centroid, pitch, and STFT. Built modular visualizations to demonstrate temporal and spectral characteristics of speech signals. ## Deep Learning: - Implemented [**Transformer**](https://github.com/nikaemami/Transformer-Models) models, including [**BERT**](https://github.com/nikaemami/Transformer-Models) for sentiment analysis of IMDB movie reviews and [**Vision Transformer (ViT)**](https://github.com/nikaemami/Transformer-Models) for FashionMNIST image classification. - Created an emotion-driven [**Music Generation**](https://github.com/nikaemami/Emotion-Driven-Music-Generation) pipeline with [**EfficientNet**](https://github.com/nikaemami/Emotion-Driven-Music-Generation) for emotion classification and [**MIDINet**](https://github.com/nikaemami/Emotion-Driven-Music-Generation) for melody generation. - Trained a [**DCGAN**](https://github.com/nikaemami/DCGAN-for-Image-Generation) to generate images of realistic clothing items from the FashionMNIST dataset. - Implemented [**Binary Segmentation**](https://github.com/nikaemami/Binary-Segmentation-of-Pedestrian-Images) of pedestrian images in PyTorch, using the [**U-Net**](https://github.com/nikaemami/Binary-Segmentation-of-Pedestrian-Images) architecture. - Performed [**Object Detection** and **Recognition**](https://github.com/nikaemami/Object-Recognition-in-Video-with-YOLOv3) in video footage using [**YOLOv3**](https://github.com/nikaemami/Object-Recognition-in-Video-with-YOLOv3). - Implemented the [**EfficientNet**](https://github.com/nikaemami/Transfer-Learning-for-EfficientNet-Network) model using **Transfer Learning** for image classification tasks. - Developed a classifier for the CIFAR-10 dataset using [**CNN** architectures](https://github.com/nikaemami/CIFAR-10-Classification-Using-MLP-and-CNN-Architectures). - Implemented **Neural Style Transfer** with the pre-trained **VGG19** model in TensorFlow. - Modified pre-trained word embedding models like **GloVe** and **Word2Vec** to perform word analogies. ## Machine Learning: - [**Speech Emotion Recognition**](https://github.com/nikaemami/Speech_Emotion_Recognition) with supervised (SVM, KNN, MLP) and unsupervised (DBSCAN, K-Means, GMM) learning models. - [**EEG Signal Processing**](https://github.com/nikaemami/ML_for_EEG_Signal_Processing) with feature extraction and supervised machine learning techniques to identify neural activation time steps. - Developed different unsupervised machine learning models like [**GMM**](https://github.com/nikaemami/Gaussian_Mixture_Models), [**SFS, SBE**](https://github.com/nikaemami/SFS_and_SBE_Models), [**JTA**](https://github.com/nikaemami/Machine-Learning-Projects), and [**PCA**](https://github.com/nikaemami/PCA_Model) models in Python. - [**Metric-Based Learning**](https://github.com/nikaemami/Metric-Based-Learning): Investigating the impact of **LMNN** and **LFDA** on k-Nearest Neighbors classifier performance. - [**Genetic Algorithms**](https://github.com/nikaemami/Local-Minimum-Optimization-Using-Genetic-Algorithms): Explored the application for local minima optimization.
