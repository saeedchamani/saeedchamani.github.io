## Deep Learning & Medical Image Processing

| Project | Summary |
|---|---|
| Brain Tumor Segmentation | Implemented advanced deep networks including 2D UNETR, Swin-Unet, U-Net3+, and U-Net++ for precise tumor segmentation. |
| Feature Extraction in Medical Imaging | Implemented U-Net architecture with popular pretrained encoders for enhanced feature extraction. |
| 3D MRI Segmentation | Developed and implemented 3D Deep Networks specifically tailored for MRI image segmentation. |
| 3D Medical Image Preprocessing | Performed arrangement and comprehensive preprocessing of 3D medical images (PET/CT, CT, and MRI). |
| Biomedical Image Denoising | Implemented noise detection and reduction using Wiener Filter, Wavelet, DCT, BM3D, Sparse Coding, NLM, and ICA. |
| Morphological Image Processing | Applied advanced morphological biomedical image processing pipelines using MATLAB. |
